# 📱 Zaphone - Aplicación de Teléfono Extensible
<img alt="Logo" src="graphics/icon.webp" width="120" />

**Zaphone** es un fork avanzado de [Fossify Phone](https://github.com/FossifyOrg/Phone) que incorpora un **sistema modular de extensiones** revolucionario para ampliar las funcionalidades de la aplicación de teléfono de Android de forma flexible y escalable.

Potencia tus llamadas con extensiones avanzadas. Zaphone redefine la experiencia móvil con privacidad incomparable, eficiencia y extensibilidad. Libre de anuncios y permisos intrusivos, está diseñado para comunicación diaria fluida y segura con poderosas capacidades de personalización.

## 🚀 **¿Por qué Zaphone?**

📱 **TU PRIVACIDAD, NUESTRA PRIORIDAD:**  
Bienvenido a Zaphone, donde tu privacidad digital es primordial. Cambia a una experiencia móvil que respeta tus datos, asegurando que tu información personal permanezca segura y privada.

⚡ **RENDIMIENTO FLUIDO:**  
Zaphone ofrece una interfaz móvil fluida y responsiva, mejorando el rendimiento de tu teléfono mientras protege tu privacidad. Experimenta una experiencia de usuario sin retrasos, optimizada para eficiencia y velocidad.

🌐 **GARANTÍA OPEN-SOURCE:**  
Con Zaphone, la transparencia está al alcance de tus dedos. Construido sobre una base de código abierto, nuestra aplicación te permite revisar nuestro código en GitHub, fomentando la confianza y una comunidad comprometida con la privacidad.

🧩 **SISTEMA DE EXTENSIONES MODULAR:**  
La característica distintiva de Zaphone: un sistema de extensiones que permite agregar funcionalidades avanzadas sin comprometer la estabilidad del núcleo de la aplicación.

## 🧩 **Sistema de Extensiones de Zaphone**

Zaphone introduce un sistema modular revolucionario que transforma tu experiencia telefónica. Cada extensión agrega capacidades específicas que van más allá de una aplicación de teléfono tradicional:

### 🎯 **Extensiones Disponibles en Zaphone v1.0**

#### 1. ✅ **Contestador IVR Inteligente**
Convierte tu Zaphone en una centralita profesional:
- **Flujos interactivos**: Crea menús de voz personalizados ("Presiona 1 para ventas, 2 para soporte...")
- **Texto-a-voz avanzado**: Respuestas automáticas naturales sin grabaciones
- **Captura DTMF**: Detecta y procesa teclas pulsadas durante la llamada
- **Gestión de flujos**: Guarda y reutiliza configuraciones para diferentes situaciones
- **Casos de uso**: Negocios, consultorios médicos, servicios de atención al cliente

#### 2. 🤖 **Asistente de IA Conversacional**
Tu asistente personal inteligente para llamadas:
- **Interacción en tiempo real**: Conversa naturalmente con los llamantes
- **Procesamiento de voz**: Entiende y responde automáticamente
- **Contexto personalizable**: Configura personalidad y conocimientos del asistente
- **Activación inteligente**: Se activa según reglas que tú defines
- **Casos de uso**: Filtrado de llamadas, información básica, programación de citas

#### 3. 📱 **Interceptor de Códigos USSD**
Control total sobre códigos especiales:
- **Interceptación `*#06#`**: Evita mostrar tu IMEI real
- **Pantallas personalizadas**: Muestra información que tú elijas
- **Seguridad mejorada**: Protege información sensible del dispositivo
- **Extensible**: Fácil agregar más códigos USSD
- **Casos de uso**: Privacidad, información corporativa, bromas inofensivas

#### 4. 🚫 **Bloqueo de Llamadas Inteligente**
Protección avanzada contra llamadas no deseadas:
- **Bloqueo por número**: Lista negra específica y precisa
- **Patrones regex**: Bloquea rangos completos (ej: todos los números que empiecen con 900)
- **Lista negra dinámica**: Agrega números sobre la marcha
- **Registro detallado**: Historial completo de llamadas bloqueadas
- **Casos de uso**: Anti-spam, telemarketing, ex-parejas molestas

### ⚙️ **Configuración Fácil de Extensiones**
1. **Abre Zaphone** en tu dispositivo
2. **Accede al menú** → **"Configuración de Extensiones"**
3. **Activa/desactiva** extensiones con un simple toque
4. **Configura individualmente** cada extensión según tus necesidades
5. **Guarda configuraciones** para diferentes perfiles de uso

> 💡 **Tip**: Puedes tener diferentes configuraciones para trabajo, casa, y situaciones especiales.

## 📦 **Instalación y Despliegue**

### 📱 **Instalación desde APK (Recomendado)**
1. **Descarga** `Zaphone-v1.0-debug.apk` desde la rama `extension-system-implementation` de este repositorio
2. **Habilita instalación** de fuentes desconocidas en Configuración → Seguridad
3. **Instala** el archivo APK en tu dispositivo Android 7.0+
4. **Configura** Zaphone como aplicación de teléfono predeterminada
5. **Accede** a "Configuración de Extensiones" desde el menú principal

### 🛠️ **Compilación desde Código Fuente**
```bash
# Clonar repositorio con sistema de extensiones
git clone -b extension-system-implementation https://github.com/gambatech/Telefono.git
cd Telefono

# Compilar APK de debug
./gradlew assembleFossDebug

# APK generado en: app/build/outputs/apk/foss/debug/
```

**Requisitos técnicos:**
- Android Studio Arctic Fox o superior
- JDK 17+
- Android SDK 34
- Gradle 8.0+

## 🔧 **Desarrollo y Arquitectura Técnica**

### 🏗️ **Arquitectura del Sistema de Extensiones**

Zaphone implementa un patrón de arquitectura modular empresarial basado en:

- **`ExtensionManager`**: Gestor central con ciclo de vida completo de extensiones
- **`BaseExtension`**: Clase abstracta que define el contrato de extensiones
- **`ExtensionRegistry`**: Sistema de registro y descubrimiento automático
- **`ExtensionSettingsActivity`**: Interfaz unificada de configuración
- **`CallStateManager`**: Coordinador de eventos de llamadas entre extensiones

### 🧩 **API para Desarrolladores - Crear Extensiones**

```kotlin
class MiExtensionEmpresarial : BaseExtension() {
    override val name = "Extensión Empresarial"
    override val description = "Funcionalidad personalizada para organizaciones"
    override val version = "1.0.0"
    
    override fun initialize(context: Context) {
        // Inicialización de recursos y configuración
    }
    
    override fun onCallStateChanged(state: CallState, phoneNumber: String?) {
        when (state) {
            CallState.INCOMING -> handleIncomingCall(phoneNumber)
            CallState.ACTIVE -> handleActiveCall()
            CallState.ENDED -> handleCallEnded()
        }
    }
    
    override fun onIncomingCall(phoneNumber: String): Boolean {
        return if (shouldHandleCall(phoneNumber)) {
            processCustomLogic(phoneNumber)
            true // Extensión maneja la llamada
        } else false
    }
    
    override fun getSettingsFragment(): Fragment? {
        return MiExtensionSettingsFragment()
    }
}
```
